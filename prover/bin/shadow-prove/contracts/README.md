# Shadow Proving
The contract required for running shadow-proving mode for morph-l2.

### Init
```
mkdir -p lib  
git clone https://github.com/foundry-rs/forge-std.git lib/forge-std
```

### Build
```
forge build  
```

## deploy
`npm install --production` + `node ./scripts/deploy.js`  
or `./deploy.sh`

### Test
```
forge test
```
